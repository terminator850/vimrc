## tab

|:--:|:--:|:--:|
|新建标签|tabnew {file_path}||
|向后换标签|tabn(ext)|<Tab>|
|向前切换标签|tabp(revious)|<S-tab>|

### 窗口大小

||命令|快捷键|
|:--:|:--:|:--:|
|窗口高度|resize (+/-5)|↑/↓|

||命令|快捷键|
|:--:|:--:|:--:|
|当前窗口打开缓存浏览|BufExplorer|<Leader>be|

|水平打开缓存浏览|BufExplorerHorizontalSplit|<Leader>bs|
|垂直打开缓存浏览|BufExplorerVerticalSplit|<Leader>bv|
